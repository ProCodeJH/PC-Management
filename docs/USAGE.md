# 사용 매뉴얼

관리자 PC에서 일상적으로 하는 일 정리.

---

## 1. 대시보드 열기

- **URL**: <http://localhost:3001>
- **로그인**: `admin` / `<비밀번호>` (초기 `admin123`, 첫날 변경 필수)
- 로그인 후 대시보드에 등록된 학생 PC 전체 grid 표시

### 카드 색 의미

| 상태 | LED | 의미 |
|---|---|---|
| 녹색 펄스 | ● (glow) | online, 최근 10초 내 heartbeat |
| 회색 | ● | offline, 5분 이상 미응답 |
| 황색 | ● | 연결 중 또는 문제 징후 |

---

## 2. 원격 뷰어 사용

### 2-1. 열기

- 대시보드에서 **online PC 카드 클릭** → 새 창에 `/remote.html?pc=NAME` 열림

### 2-2. 상단 hover 툴바

마우스를 화면 상단으로 가져가면 툴바 표시 (2.5초 idle 자동 숨김):

```
←  PC-NAME  ●연결됨   지연 12ms  FPS 24  해상도 1920×1080   전체화면  제어 OFF  ✕
```

### 2-3. 단축키

| 키 | 동작 |
|---|---|
| `F` | 전체화면 토글 |
| `Ctrl+Shift+M` | 원격 제어 ON/OFF |
| `Esc` | 전체화면 해제 |
| 마우스 상단 이동 | 툴바 표시 |

### 2-4. 원격 제어 (ON 상태)

- 마우스 클릭/이동/드래그 → 학생 PC에 동일 동작
- 키보드 입력 → 학생 PC에 전달 (Win/Meta 키 포함)
- 스크롤 → 학생 PC 스크롤
- 우클릭 컨텍스트 메뉴 → 학생 PC에 표시

### 2-5. 클립보드 양방향

- 관리자 PC 창에서 **Ctrl+C** → 자동으로 학생 PC 클립보드에 반영
- 학생 PC 클립보드 변경 → 관리자 뷰어 창에 자동 반영

## 3. 파일 전송 (드래그 & 드롭)

### 학생 PC 바탕화면으로 파일 보내기

1. 원격 뷰어 창 열고
2. **파일을 드래그해서 창 위로 가져오기**
3. 드롭존 overlay 표시: "학생 PC 바탕화면으로 업로드"
4. 놓으면 자동 업로드
5. 진행바 토스트 표시 (파일명, 진행률, MB 크기)
6. 완료 시 녹색 토스트: "학생 PC 바탕화면에 저장됨"

### 제약

- 최대 **100MB** / 파일
- 동시 여러 파일 가능 (하나씩 순차 업로드)
- 실패 시 빨간 토스트에 사유 표시

### 저장 위치

학생 PC의 `C:\Users\<현재 로그인 유저>\Desktop\` 에 저장. 같은 이름 파일이 있으면 자동으로 `(1)`, `(2)` 숫자 추가.

## 4. 사이트 차단

### 4-1. 차단 추가

대시보드 사이드바 → **차단 설정** 또는 전역 버튼 → 차단 사이트 모달:

- 입력란에 도메인 (`youtube.com`) 또는 URL 입력 → 엔터
- 25개 서브도메인 변형 자동 생성 (예: `youtube.com`, `www.youtube.com`, `m.youtube.com`, `youtu.be`, ...)
- 모든 online 학생 PC에 즉시 hosts 파일 갱신

### 4-2. 차단 해제

- 목록에서 ✕ 버튼 클릭
- "전체 해제" 버튼으로 일괄 해제 가능

### 4-3. 확인

학생 PC에서 `ping youtube.com` → `127.0.0.1` 응답 오면 차단 동작 중.

## 5. 프로그램 차단 / 시험 모드

### 5-1. 프로그램 차단 (개별)

- PC 카드 우클릭 → `프로그램 차단`
- 실행 중 프로세스 목록 표시 → 차단할 것 선택
- 또는 수동으로 프로세스명 입력 (예: `LeagueOfLegends.exe`)

### 5-2. 시험 모드 (전체 PC 동시)

- 상단 `시험 모드` 버튼
- 시작 시: 모든 user-space 프로세스 종료 (chrome, notepad, cmd 등). **시스템 프로세스 보호** (`explorer.exe`, `svchost.exe` 등은 건드리지 않음).
- 종료 시: 차단 해제, 학생들 자유 사용 가능

---

## 6. CCTV 모드 (전체 동시 감시)

- 상단 `CCTV` 버튼
- 모든 online PC의 thumbnail grid (3fps, 저품질) 동시 표시
- 특정 PC 클릭 → 그 PC의 고품질 원격 뷰어 전환

대역폭 주의: 30대 동시 스트리밍은 WiFi 혼잡 유발 가능. **유선 LAN 권장**.

## 7. 출석 관리

자동으로 online 시간 추적 → `사이드바 → 출석`에서 확인:

- 날짜별 등원/하원 시각
- CSV 다운로드
- 학생별 누적 시간

## 8. 기타 기능

| 기능 | 위치 | 설명 |
|---|---|---|
| WOL | PC 카드 우클릭 → 깨우기 | MAC 기반 원격 부팅 (동일 서브넷) |

| 메시지 전송 | PC 카드 우클릭 → 메시지 | 학생 PC에 팝업 알림 |
| 에이전트 업데이트 | 사이드바 → 에이전트 | 서버의 최신 agent.js 재배포 |
| 일괄 명령 | 상단 → 전체 명령 메뉴 | 모든 PC에 한번에 종료/재부팅/잠금 |
| 스크린샷 캡처 | PC 카드 → 카메라 아이콘 | 지금 화면 한 장 저장 |

## 9. 자주 묻는 것

**Q. 원격 제어 중인데 학생이 마우스를 움직이면?**  
A. 경쟁 상태. 관리자와 학생 중 마지막 입력이 이김. 수업 중엔 `Ctrl+Shift+M`로 제어 OFF → 화면만 봄.

**Q. 화면이 뚝뚝 끊겨요.**  
A. 뷰어 툴바의 FPS 확인. 30fps 이상이면 정상. 낮으면 LAN 혼잡 or 학생 PC CPU 100%. [TROUBLESHOOTING.md](TROUBLESHOOTING.md).

**Q. 파일 업로드가 실패해요.**  
A. 100MB 초과? 학생 PC 로그인 상태 (Lock 화면이면 Desktop 접근 불가). DIAGNOSE-AGENT.bat로 확인.

**Q. 여러 PC 원격 동시?**  
A. 가능. 각 PC별 새 창으로 `/remote.html?pc=NAME` 열면 됨. 듀얼 모니터에서 유용.
